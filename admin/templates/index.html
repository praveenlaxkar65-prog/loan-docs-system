Quick API Reference

POST /api/v1/applications/create

Create application → get app_id | Header: X-API-Key

POST /api/v1/upload

Upload documents (multipart) | application_id + files

GET /api/v1/applications/{id}

Status + extracted data + checklist

GET /api/v1/applications/{id}/pdf

Download final PDF
